# Roadmap

## Near term

- Add more export formats
- Improve Windows notification polish
- Refine Linux and macOS integration
- Improve empty, error, and success states

## Mid term

- Add small recent-items history
- Optional tray mode
- Better file naming strategies
- Drag-and-drop image file support

## Long term

- Packaging and installers for each platform
- Smarter image type detection and export conversion
- Optional automation hooks for power users

## Product rule

Every roadmap item should be judged against one question:

> Does this make the core save flow faster, cleaner, and less annoying?

If the answer is no, it probably does not belong in PICO.
